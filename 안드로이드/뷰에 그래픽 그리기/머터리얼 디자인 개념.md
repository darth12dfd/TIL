# 머터리얼 디자인

* 머터리얼 디자인은 구글이 안드로이드에 적용한 디자인 트렌드 중의 하나이다. 좀더 단순하고 직관적인 디자인을 할 수 있게 도입한 것이ㅏㄷ.

* 머터리얼 디자인은 API 21부터 도입되었으며 종합적인 디자인 지침으로 이해할 수 있다.
