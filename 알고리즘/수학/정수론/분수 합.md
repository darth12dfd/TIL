# 분수 합

## 출처

* https://www.acmicpc.net/problem/1735

## 문제

* 분수 A/B는 분자가 A, 분모가 B인 분수를 의미한다. A와 B는 모두 자연수라고 하자.

* 두 분수의 합 또한 분수로 표현할 수 있다. 두 분수가 주어졌을 때, 그 합을 기약분수의 형태로 구하는 프로그램을 작성하시오. 기약분수란 더 이상 약분되지 않는 분수를 의미한다.

## 입력

* 첫째 줄과 둘째 줄에, 각 분수의 분자와 분모를 뜻하는 두 개의 자연수가 순서대로 주어진다. 입력되는 네 자연수는 모두 30,000 이하이다.

## 출력

* 첫째 줄에 구하고자 하는 기약분수의 분자와 분모를 뜻하는 두 개의 자연수를 빈 칸을 사이에 두고 순서대로 출력한다.

## 입력 예제 1

```
2 7
3 5
```

## 출력 예제 1

```
31 35
```

## 풀이

* 유클리드 호제법을 활용하는 문제. 분모와 분자 값을 곱해서 구한 다음 이를 둘 사이의 최대 공약수로 나눠서 그 값을 출력하는 문제이다. 기본적은 유클리드 호제법을 활용하면 무난하게 풀 수 있는 문제이다.

## 소스코드

```kotlin
import java.io.BufferedReader
import java.io.InputStreamReader
import java.util.StringTokenizer

fun main() = with(BufferedReader(InputStreamReader(System.`in`))) {

    var st = StringTokenizer(readLine())

    //분수 A/B
    val a1 = st.nextToken().toInt()
    val b1 = st.nextToken().toInt()

    st = StringTokenizer(readLine())

    //분수 A/B
    val a2 = st.nextToken().toInt()
    val b2 = st.nextToken().toInt()

    //분모, 분자 구하기
    var b3 = b1 * b2
    var a3 = (a1*b2) + (a2*b1)

    //분모, 분자의 최대 공약수 구하기(유클리드 호제법)
    fun gcd(a: Int, b: Int): Int {
        var num1 = a
        var num2 = b
        while (num2 != 0) { // num2가 0이 될 때까지 반복
            val temp = num1 % num2 // 나머지를 임시 저장
            num1 = num2           // 작은 수(num2)를 큰 수(num1)로
            num2 = temp           // 나머지를 작은 수(num2)로
        }
        return num1 // num1이 최대공약수
    }

    var gcd = gcd(a3,b3)

    println("${a3/gcd} ${b3/gcd}")

    close()
}
```
