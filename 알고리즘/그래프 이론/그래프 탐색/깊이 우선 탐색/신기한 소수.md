# 신기한 소수

## 출처

* https://www.acmicpc.net/problem/2023

## 문제

* 수빈이가 세상에서 가장 좋아하는 것은 소수이고, 취미는 소수를 가지고 노는 것이다. 요즘 수빈이가 가장 관심있어 하는 소수는 7331이다.

* 7331은 소수인데, 신기하게도 733도 소수이고, 73도 소수이고, 7도 소수이다. 즉, 왼쪽부터 1자리, 2자리, 3자리, 4자리 수 모두 소수이다! 수빈이는 이런 숫자를 신기한 소수라고 이름 붙였다.

* 수빈이는 N자리의 숫자 중에서 어떤 수들이 신기한 소수인지 궁금해졌다. N이 주어졌을 때, 수빈이를 위해 N자리 신기한 소수를 모두 찾아보자.

## 입력

* 첫째 줄에 N(1 ≤ N ≤ 8)이 주어진다.

## 출력

* N자리 수 중에서 신기한 소수를 오름차순으로 정렬해서 한 줄에 하나씩 출력한다.

## 입력 예제 1

```
4
```

## 출력 예제 1

```
2333
2339
2393
2399
2939
3119
3137
3733
3739
3793
3797
5939
7193
7331
7333
7393
```

## 풀이

* 첫 자리는 소수가 될 수 있는 숫자(2, 3, 5, 7)만 가능.

* 뒤에 붙일 수 있는 후보는 홀수(1, 3, 7, 9)만 검사.

* 매 단계마다 소수 판별 → 유효하면 재귀(DFS)로 확장.

* 길이가 N이면 출력.

## 소스코드

```kotlin
package graph.dfs

import java.io.BufferedReader
import java.io.InputStreamReader
import kotlin.math.sqrt

fun main() {
    val reader = BufferedReader(InputStreamReader(System.`in`))
    val N = reader.readLine().trim().toInt()

    // 첫 자리 후보 (소수만)
    val firstPrimes = listOf(2, 3, 5, 7)

    // 뒤에 붙일 후보 (짝수 및 5 제외)
    val nextDigits = listOf(1, 3, 7, 9)

    val sb = StringBuilder()

    // 소수 판별 함수
    fun isPrime(x: Int): Boolean {
        if (x < 2) return false
        val r = sqrt(x.toDouble()).toInt()
        for (i in 2..r) {
            if (x % i == 0) return false
        }
        return true
    }

    // DFS 탐색
    fun dfs(current: Int, depth: Int) {
        if (depth == N) {
            sb.append(current).append('\n')
            return
        }
        for (d in nextDigits) {
            val next = current * 10 + d
            if (isPrime(next)) {
                dfs(next, depth + 1)
            }
        }
    }

    // DFS 시작
    for (p in firstPrimes) {
        dfs(p, 1)
    }

    print(sb.toString())
}

```
