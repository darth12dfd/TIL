# 안전 영역

## 출처

* https://www.acmicpc.net/problem/2468

## 문제

* 재난방재청에서는 많은 비가 내리는 장마철에 대비해서 다음과 같은 일을 계획하고 있다. 먼저 어떤 지역의 높이 정보를 파악한다. 그 다음에 그 지역에 많은 비가 내렸을 때 물에 잠기지 않는 안전한 영역이 최대로 몇 개가 만들어 지는 지를 조사하려고 한다. 이때, 문제를 간단하게 하기 위하여, 장마철에 내리는 비의 양에 따라 일정한 높이 이하의 모든 지점은 물에 잠긴다고 가정한다.

* 어떤 지역의 높이 정보는 행과 열의 크기가 각각 N인 2차원 배열 형태로 주어지며 배열의 각 원소는 해당 지점의 높이를 표시하는 자연수이다. 예를 들어, 다음은 N=5인 지역의 높이 정보이다.


| 6	| 8	| 2	| 6	| 2 |
| ---- | ---- | ---- | ---- | ---- |
| 3	| 2	| 3	| 4	| 6 |
| 6	| 7	| 3	| 3	| 2 |
| 7	| 2	| 5	| 3	| 6 |
| 8	| 9	| 5	| 2	| 7 |

* 이제 위와 같은 지역에 많은 비가 내려서 높이가 4 이하인 모든 지점이 물에 잠겼다고 하자. 이 경우에 물에 잠기는 지점을 회색으로 표시하면 다음과 같다.

* 이와 같이 장마철에 내리는 비의 양에 따라서 물에 잠기지 않는 안전한 영역의 개수는 다르게 된다. 위의 예와 같은 지역에서 내리는 비의 양에 따른 모든 경우를 다 조사해 보면 물에 잠기지 않는 안전한 영역의 개수 중에서 최대인 경우는 5임을 알 수 있다.

* 어떤 지역의 높이 정보가 주어졌을 때, 장마철에 물에 잠기지 않는 안전한 영역의 최대 개수를 계산하는 프로그램을 작성하시오.

## 입력

* 첫째 줄에는 어떤 지역을 나타내는 2차원 배열의 행과 열의 개수를 나타내는 수 N이 입력된다. N은 2 이상 100 이하의 정수이다. 둘째 줄부터 N개의 각 줄에는 2차원 배열의 첫 번째 행부터 N번째 행까지 순서대로 한 행씩 높이 정보가 입력된다. 각 줄에는 각 행의 첫 번째 열부터 N번째 열까지 N개의 높이 정보를 나타내는 자연수가 빈 칸을 사이에 두고 입력된다. 높이는 1이상 100 이하의 정수이다.

## 출력

* 첫째 줄에 장마철에 물에 잠기지 않는 안전한 영역의 최대 개수를 출력한다.

## 입력 예제 1

```
5
6 8 2 6 2
3 2 3 4 6
6 7 3 3 2
7 2 5 3 6
8 9 5 2 7
```

## 출력 예제 1

```
5
```

## 입력 예제 2

```
7
9 9 9 9 9 9 9

9 1 8 7 8 1 9
9 2 7 9 7 2 9

```

## 출력 예제 2

```
6
```

## 참고

* 아무 지역도 물에 잠기지 않을 수도 있다.

## 풀이

* 비의 양을 0부터 최고 높이(maxHeight)까지 바꿔 가면서 잠기지 않는 칸들로 이루어진 연결된 영역(상하좌우)을 세고, 그 중 가장 큰 영역 개수를 구하는 문제

1. 입력을 받으면서 maxHeight를 같이 구함.

2.  비의 양 h를 0부터 maxHeight까지 반복:

    - 매번 방문(visited) 배열을 새로 만듦.

    - 모든 칸을 돌면서 아직 방문 안 했고, height > h인 칸이면 BFS/DFS로 같은 영역을 모두 방문 처리하고 안전 영역 개수 +1.

3. 모든 h에 대해 구한 안전 영역 개수 중 최댓값을 출력.

## 소스코드

```kotlin
package graph.bfs

import java.io.BufferedReader
import java.io.InputStreamReader
import java.util.StringTokenizer
import java.util.ArrayDeque

fun main() {
    val br = BufferedReader(InputStreamReader(System.`in`))

    val n = br.readLine().trim().toInt()
    val map = Array(n) { IntArray(n) }

    var maxHeight = 0

    for (i in 0 until n) {
        val st = StringTokenizer(br.readLine())
        for (j in 0 until n) {
            val h = st.nextToken().toInt()
            map[i][j] = h
            if (h > maxHeight) maxHeight = h
        }
    }

    // 방향: 상, 하, 좌, 우
    val dx = intArrayOf(-1, 1, 0, 0)
    val dy = intArrayOf(0, 0, -1, 1)

    var answer = 0

    // 비의 양 h를 0부터 maxHeight까지 전부 시도
    for (h in 0..maxHeight) {
        val visited = Array(n) { BooleanArray(n) }
        var safeCount = 0

        for (i in 0 until n) {
            for (j in 0 until n) {
                // 아직 방문 안 했고, 현재 비 높이 h보다 높은 칸이면 새로운 영역 시작
                if (!visited[i][j] && map[i][j] > h) {
                    safeCount++

                    // BFS로 연결된 영역 전체 방문
                    val q: ArrayDeque<Pair<Int, Int>> = ArrayDeque()
                    q.add(Pair(i, j))
                    visited[i][j] = true

                    while (q.isNotEmpty()) {
                        val (x, y) = q.removeFirst()

                        for (dir in 0 until 4) {
                            val nx = x + dx[dir]
                            val ny = y + dy[dir]

                            if (nx in 0 until n && ny in 0 until n) {
                                if (!visited[nx][ny] && map[nx][ny] > h) {
                                    visited[nx][ny] = true
                                    q.add(Pair(nx, ny))
                                }
                            }
                        }
                    }
                }
            }
        }

        if (safeCount > answer) {
            answer = safeCount
        }
    }

    println(answer)
}

```
