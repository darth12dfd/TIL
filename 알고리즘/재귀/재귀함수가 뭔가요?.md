# 재귀함수가 뭔가요?

## 출처

* https://www.acmicpc.net/problem/17478

## 문제

* 평소에 질문을 잘 받아주기로 유명한 중앙대학교의 JH 교수님은 학생들로부터 재귀함수가 무엇인지에 대하여 많은 질문을 받아왔다.

* 매번 질문을 잘 받아주셨던 JH 교수님이지만 그는 중앙대학교가 자신과 맞는가에 대한 고민을 항상 해왔다.

* 중앙대학교와 자신의 길이 맞지 않다고 생각한 JH 교수님은 결국 중앙대학교를 떠나기로 결정하였다.

* 떠나기 전까지도 제자들을 생각하셨던 JH 교수님은 재귀함수가 무엇인지 물어보는 학생들을 위한 작은 선물로 자동 응답 챗봇을 준비하기로 했다.

* JH 교수님이 만들 챗봇의 응답을 출력하는 프로그램을 만들어보자.
 
## 입력

* 교수님이 출력을 원하는 재귀 횟수 N(1 ≤ N ≤ 50)이 주어진다.

## 출력

* 출력 예시를 보고 재귀 횟수에 따른 챗봇의 응답을 출력한다.

## 입력 예제 1

```
2
```

## 풀이

* 기본적인 재귀 문제이다. 재귀 함수를 호출할 때마다 언더바를 4개씩 추가해 주고, 언더바 뒤에 재귀로 출력할 문장을 출력해주면 된다.

## 소스코드

```kotlin
package recursion

import java.io.BufferedReader
import java.io.InputStreamReader

fun main() = with(BufferedReader(InputStreamReader(System.`in`))){
    val N = readLine().toInt()//출력을 원하는 재귀의 횟수(1 이상 50 이하의 자연수)

    var underbar = ""

    println("어느 한 컴퓨터공학과 학생이 유명한 교수님을 찾아가 물었다.")

    fun recur(n:Int){
        var line = underbar

        //재귀 함수 종료
        if(n == 0){
            println("$line" + "\"재귀함수가 뭔가요?\"")
            println("$line" + "\"재귀함수는 자기 자신을 호출하는 함수라네\"")
            println("$line" + "라고 답변하였지.")

            return
        }

        //재귀 함수 반복
        println("$line" + "\"재귀함수가 뭔가요?\"")
        println("$line" + "\"잘 들어보게. 옛날옛날 한 산 꼭대기에 이세상 모든 지식을 통달한 선인이 있었어.")
        println("$line" + "마을 사람들은 모두 그 선인에게 수많은 질문을 했고, 모두 지혜롭게 대답해 주었지.")
        println("$line" + "그의 답은 대부분 옳았다고 하네. 그런데 어느 날, 그 선인에게 한 선비가 찾아와서 물었어.\"")


        //줄 생성
        underbar += "____"

        recur(n-1)

        //맨 마지막
        println("$line" + "라고 답변하였지.")
    }

    recur(N)


    close()
}
```
