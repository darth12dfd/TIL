# 구간 나누기 2

## 출처

* https://www.acmicpc.net/problem/13397

## 문제

* N개의 수로 이루어진 1차원 배열이 있다. 이 배열을 M개 이하의 구간으로 나누어서 구간의 점수의 최댓값을 최소로 하려고 한다. 구간은 다음과 같은 조건을 만족해야 한다.

```
하나의 구간은 하나 이상의 연속된 수들로 이루어져 있다.
배열의 각 수는 모두 하나의 구간에 포함되어 있어야 한다.
```

* 구간의 점수란 구간에 속한 수의 최댓값과 최솟값의 차이이다.

* 예를 들어, 배열이 [1, 5, 4, 6, 2, 1, 3, 7] 이고, M = 3인 경우가 있다.

* 이때, [1, 5], [4, 6, 2], [1, 3, 7]로 구간을 나누면 각 구간의 점수는 4, 4, 6점이 된다. 이때, 최댓값은 6점이다.

* 만약, [1, 5, 4], [6, 2, 1], [3, 7]로 구간을 나누었다면, 각 구간의 점수는 4, 5, 4점이 되고, 이때 최댓값은 5점이 된다.

* 두 경우 중에서 최댓값이 최소인 것은 5점인 것이고, 5점보다 최댓값을 작게 만드는 방법은 없다.

* 배열과 M이 주어졌을 때, 구간의 점수의 최댓값의 최솟값을 구하는 프로그램을 작성하시오.

## 입력

* 첫째 줄에 배열의 크기 N과 M이 주어진다. (1 ≤ N ≤ 5,000, 1 ≤ M ≤ N)

* 둘째 줄에 배열에 들어있는 수가 순서대로 주어진다. 배열에 들어있는 수는 1보다 크거나 같고, 10,000보다 작거나 같은 자연수이다.

## 출력

* 첫째 줄에 구간의 점수의 최댓값의 최솟값을 출력한다.

## 입력 예제 1

## 소스코드

```kotlin
import java.io.BufferedReader
import java.io.InputStreamReader
import java.util.StringTokenizer
import kotlin.math.max
import kotlin.math.min

fun main() {

    val br = BufferedReader(InputStreamReader(System.`in`))
    val st = StringTokenizer(br.readLine())

    // N : 배열 길이
    // M : 만들 수 있는 최대 구간 개수
    val n = st.nextToken().toInt()
    val m = st.nextToken().toInt()

    // 수열 입력
    val arr = br.readLine().split(" ").map { it.toInt() }

    /*
        이 문제는 "구간 점수의 최대값"을 최소화하는 문제

        구간 점수 = (구간의 최대값 - 구간의 최소값)

        찾는 값:
        -> 모든 구간 점수 중 최대값의 최소

        이런 형태는 보통 "파라메트릭 서치 (이분 탐색)"로 해결한다.
     */

    // 가능한 최소 점수
    var left = 0

    // 가능한 최대 점수
    // 배열 전체를 하나의 구간으로 잡을 경우
    var right = arr.max() - arr.min()

    /*
        limit : 우리가 허용할 "구간 점수 최대값"

        이 limit 안에서
        배열을 M개 이하 구간으로 나눌 수 있는지 검사한다.
     */
    fun canDivide(limit: Int): Boolean {

        // 현재까지 사용한 구간 개수
        var count = 1

        // 현재 구간의 최소/최대
        var minVal = arr[0]
        var maxVal = arr[0]

        for (i in 1 until n) {

            // 현재 구간에 원소를 추가하면서
            min/max 업데이트
            minVal = min(minVal, arr[i])
            maxVal = max(maxVal, arr[i])

            /*
                구간 점수 = max - min

                만약 limit을 초과하면
                -> 현재 원소부터 새로운 구간 시작
             */
            if (maxVal - minVal > limit) {

                count++

                // 새 구간 시작
                minVal = arr[i]
                maxVal = arr[i]
            }
        }

        // 구간 수가 M 이하이면 가능
        return count <= m
    }

    /*
        이분 탐색

        mid = 허용 가능한 구간 점수의 최대값
     */
    while (left <= right) {

        val mid = (left + right) / 2

        if (canDivide(mid)) {
            // mid 값으로도 구간을 M개 이하로 만들 수 있음
            // -> 더 작은 값도 가능한지 확인
            right = mid - 1
        } else {
            // 구간이 너무 많이 필요함
            // -> limit을 늘려야 함
            left = mid + 1
        }
    }

    // 최소 가능한 구간 점수 최대값
    println(left)
}
```
